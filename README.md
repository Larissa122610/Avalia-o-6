# Avalia-o-6
A Galeria Online de Larissa Vaz Terra reúne obras originais e coleções exclusivas disponíveis para compra direta do ateliê. Cada peça expressa cor, gesto e emoção, unindo sensibilidade e autenticidade. Mais que uma vitrine, o site é um convite para viver a arte — decorar, colecionar e se conectar com criações únicas que transformam espaços.
